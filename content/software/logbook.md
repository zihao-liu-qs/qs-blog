Logbook 是一个轻量的开发日志工具，帮你在编码间隙记录决策、困惑和灵感。它不是 issue tracker，也不是文档工具——它更像一本工程师的私人日记。

## 计划功能

- 按项目/日期组织日志条目
- 与 Git commit 关联
- 本地加密存储
- 导出为 Markdown
