Etch — 纯文本笔记

文件夹 + Markdown，就是你需要的全部

¥48 购买

Etch 不是又一款笔记应用。它只做一件事：让你的 Markdown 文件在本地文件夹里，打开就用，关掉就走。没有账户，没有同步订阅，没有数据库。

功能

  • 直接读写本地文件夹里的 .md 文件
  • 实时预览，所见即所得切换
  • 双链语法支持（[[文件名]]）
  • iCloud Drive 天然同步
  • 无障碍全键盘操作

系统要求

macOS 13 / iOS 16 及以上
